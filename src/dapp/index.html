  • CONTRACT OWNER/ADMIN

    Only the contract owner and verified admins have access to this card. This control center allows for the upgrade and pausing of the application smart contract, (which is responsible for the interaction between the UI and the persistent data contract) should anything malicious surface in the future.

    In the event of a pause, sending and receiving ether will not be possible until unpaused.

    In the event of a contract upgrade, all data and funds in the contract will be untouched and safe from corruption of any kind. However, some features of this application might have changed for improved user experience.
